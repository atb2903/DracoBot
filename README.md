# DracoBot
A personal discord bot

This is the tutorial branch, in which I follow along with Discord's tutorial for bulding a Discord bot

* https://docs.discord.com/developers/quick-start/getting-started

## Setup

### Requirements

* NPM  
* ngrok
* Node.js

From the project root, run the following:

* `npm install`
* `npm run register`

## Run

From the project root, run the following:

* `npm run start`
* `ngrok http 3000`
